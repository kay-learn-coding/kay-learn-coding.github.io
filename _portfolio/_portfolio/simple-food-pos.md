Built a modular, Object-Oriented Programming (OOP) desktop ordering application featuring a graphical user interface (GUI) designed to bridge front-of-house hospitality operations directly with automated inventory management.

### Key Technical Achievements:
* **Object-Oriented Design:** Leveraged encapsulation and class inheritance structures (`MenuItem` base class with specialized `Food` and `Drink` subclasses).
* **GUI Development:** Engineered an interactive interface using Python's `Tkinter` library, utilizing reactive layout frames, event loops, and dynamic lambda callbacks.
* **Robust Input Validation:** Implemented strict error handling via `try/except` exception structures and `ValueError` traps.
* **Persistent Storage Automation:** Integrated an atomic data synchronization layer using the `json` and `os` path modules, enabling absolute directory mapping.
